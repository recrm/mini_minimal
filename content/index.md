This is a minimal computing website generator. It combines markdown files with HTML templates to create static websites.

Learn more [about this project](about.html) or check out the [picture gallery](pictures.html).

## Features

- **Simple**: Just markdown files and a template
- **Fast**: Generates static HTML files
- **Flexible**: Customize templates to match your style
- **Minimal**: No complex build systems or dependencies

## How It Works

1. Write your content in markdown files
2. Customize the HTML template
3. Run the generator
4. Deploy the static HTML files
